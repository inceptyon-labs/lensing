# @lensing/display

SvelteKit display app for the Lensing kiosk. Renders a zone-based layout with plugin widgets.

## Zones

- `top-bar` — top navigation / clock
- `left-col` — sidebar widgets
- `center` — primary content area
- `right-col` — secondary sidebar
- `bottom-bar` — status / footer

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
